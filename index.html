Conditionals in Javascript

Conditionals are statements that help your program make decisions. They check if something is true or false, and then execute different code depending on the result.

1.if Statement
Used to run code only if a condition is true. eg: //let age = 18; //if (age >= 18) { //console.log("You are an adult."); }

2. if.. else Runs one block if the condition is true, and another block if itâ€™s false. eg: let age = 16; if (age >= 18) { console.log("You are an adult."); } else { console.log("You are a minor."); }

3. if else if else ladder
Used when you have multiple conditions to check.

4. Switch Statement:
Used when you need to compare one value to many possible cases.

5.TERNANRY OPERATOR (Short if... else)

A quick way to write conditionals in one line. eg: let age = 20; //let result = (age >= 18) ? "Adult" : "Minor"; //console.log(result); // output : Adult

Note: 1. Always use break to stop the code from running into the next case. 2.default runs if none of the cases match.
